# Robert Jenkinson Alvarez Research Site

Static personal research site built from the Clarity template structure and content migrated from `/Users/robjenkinson/Documents/Website`.

## Local Preview

Serve this folder with any static file server, for example:

```sh
python3 -m http.server 1234
```

Then open `http://localhost:1234/index.html`.

## Attribution

This site uses the open-source [Clarity Template](https://shikun.io/projects/clarity). The footer keeps the required template link-back.
