thesis: We fit an arbitrage-carrying forward-discounted call surface directly in price space, enforcing static no-arbitrage through sparse linear operators, and a local fog layer is used where quotes are inconsistent.
problem: Quoted option surfaces are noisy and can violate monotonicity, convexity, and calendar consistency. If the fitted object is not the arbitrage-carrying quantity itself, violations can be hidden until late in the pipeline.
coreIdea: Fit the forward-discounted call object `C_f(m, tau)` directly in price space. Use endpoint-preserving warps on moneyness and maturity so polynomial approximation behaves better near boundaries and sparse data regions.
method: Expand the warped surface in a tensor Chebyshev basis, impose static no-arbitrage as sparse linear inequalities on a collocation grid, and solve a convex quadratic program as the global backbone. Apply the Hamiltonian fog only as a local post-fit patch in inconsistent regions.
results: In calm regimes, the calibrated surface reaches 98-99% inside-spread coverage with below 1% static no-arbitrage violations, while preserving an auditable fit.

## Problem

The global fit stays convex: prices, derivatives, constraints, and regularizers all act on the coefficient vector.

## What changes

The fit is built for surfaces that must stay no-arbitrage under calibration, not just pass through mids.

## Technical Notes

- The QP step is the main structural engine; the fog module is intentionally local and secondary.
- Constraint sparsity keeps enforcement scalable while preserving interpretation.
